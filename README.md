# Melee Web

## Setup Instructions

1. **Clone the repository**:
   ```bash
   git clone https://github.com/ludicoloblue/melee-web.git
   ```

2. **Navigate into the directory**:
   ```bash
   cd melee-web
   ```

3. **Install dependencies**:
   ```bash
   npm install
   ```

4. **Start the development server**:
   ```bash
   npm start
   ```
